# AudioVerse English

AudioVerse English is a documentation-first software project for an offline-first, audio-based English learning platform designed for blind and low-vision learners.

This repository intentionally starts without application source code. The first development milestone is to complete professional-grade software engineering documentation before any Flutter project is created.

## Repository Status

- Current mode: Documentation and planning
- Application code: Not started
- Target frontend: Flutter
- Target local database: SQLite
- Optional cloud sync: Supabase
- Primary accessibility targets: TalkBack, VoiceOver, WCAG, gesture navigation, high contrast mode, blank screen mode
